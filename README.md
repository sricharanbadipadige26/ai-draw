# AI Draw 🎨

AI Draw is a Python-based virtual drawing application that allows users to draw on a digital canvas using only hand gestures captured through a webcam. The project uses computer vision and hand tracking to provide a touch-free drawing experience.

## Features
- ✋ Real-time hand gesture recognition
- 🎨 Draw using finger movements
- 🌈 Multiple color selection
- 🧽 Eraser mode
- 🗑️ Clear canvas
- 📷 Webcam-based interaction

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

## How It Works
The application detects hand landmarks using MediaPipe and tracks the index finger to draw on a virtual canvas. Different gestures are used to switch between drawing, erasing, and selecting colors.
